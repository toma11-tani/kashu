【画像ファイルの配置について】

このフォルダに以下の2枚の画像を配置してください：

1. hero_shop.jpg
   - 店舗外観写真
   - 用途: トップページのヒーロー画像、店舗情報ページ
   - 推奨サイズ: 1920×1080px以上（横長）
   - 形式: JPEG

2. menu_luroufan.jpg
   - 魯肉飯（ルーローハン）の写真
   - 用途: トップページの名物紹介、メニューページ

   - 形式: JPEG

【注意事項】
- ファイル名は必ず上記の通りにしてください（大文字小文字も区別されます）
- 画像は web 向けに最適化（圧縮）することを推奨します
- 画質は70〜85%程度で十分です
- ファイルサイズは1枚あたり500KB以下を目安にしてください
